📊 Sales Analysis Project

This project analyzes sales data to identify revenue trends, top-selling products, and regional sales performance using Python, SQL, and Power BI.

The dataset contains 5000+ sales records including product details, customer information, city, quantity, price, and order dates. The project demonstrates the complete data analytics workflow, from data generation and SQL analysis to interactive dashboard visualization.

🚀 Project Objective

The objective of this project is to analyze sales data and extract business insights that help understand:

Sales trends over time

Best-selling products

Revenue contribution by city

Category-wise performance

Customer purchasing patterns

🛠 Tools & Technologies

Python (Pandas) – Dataset generation and analysis

MySQL – Data querying and analysis using SQL

Power BI – Interactive data visualization dashboard

CSV – Dataset storage format
